# Teste de usabilidade

> Testes com usuários reais ou simulados para identificar fricções.
> 
- Pode fazer com 3 a 5 pessoas usando um protótipo no Figma.
- Perguntas abertas + observações (ex: “Eles conseguem montar um prato de 3 blocos sem ajuda?”).
- Documente feedback e o que foi ajustado depois disso.
